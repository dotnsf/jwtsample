# JWT Sample

Simple sample for JWT(Json Web Token) with Express-Session


## APIs

- GET /(index.html)

    - Show index page.

- GET /login.html

    - Show login page.

- GET /logout

    - Logout, and show index page.

- POST /login

    -  Login with id and password

- GET /items

    - Get items list

- POST /item

    - Create new item, and add to items list
